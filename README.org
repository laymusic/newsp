This directory is for the code to produce the html files that will be
the new serpentpublications.org website.

It currently writes them to ~/public_html/newsp...

The database will be the musicpublish mysql database on
serpentpublications.org

piecepages/ will contain a file per piece, named <number>.html, where
number is the id of the piece in the database.

composerpages/ will contain a file per composer, named <number>.html,
where number is the composer_id of the composer in the database.

bookpages/ will contain a file per book, named <number>.html, where
number is the book_id of the book in the database.

sorts/ will contain files of pieces sorted by some method, for
instance bycomposer.html or bylastmodified.html
